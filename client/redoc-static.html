Маркетинг

Здесь обрабатываются все запросы, связанные с маркетингом

Генерация маркетингового контента

Authorizations:
BearerAuth
Request Body schema: application/json
required
topic
string

Описание, которое дал пользователь для темы поста

amount
integer

Количество вариантов, которые надо сгенерировать

style
string

Опциональный параметр про стиль, в котором будет сгенерирован ответ. Может быть пустым, тогда надо использовать обычный системный промпт

contentType
string (MarketingContentType)
Enum: "outodoor_ad" "social_media" "online_ad" "brochure" "email"
