## Настройка базы данных

1. Установите PostgreSQL локально
2. Введите в терминал команду `pip install python-dotenv`.
3. Создайте файл `.env` в соответствии с `.env.example`.
4. Заполните своими значениями. Файл `.env` локален на вашем компьютере.
5. Для создания БД запустите файл `db\scripts\init_db.py`.
6. Все функции размещены в файле `db\scripts\operations_db.py`.
В случае неполадок БД можно удалить `db\scripts\delete_db.py`.
